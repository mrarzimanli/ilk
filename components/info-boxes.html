Usage Guide

How to use the info-box component

Base Structure

The info-box requires this HTML structure:

<div class="info-box info-box--blue"> <div class="info-box__icon"> <i class="ri-information-line"></i> </div> <div class="info-box__content"> <div class="info-box__title">Title</div> <div class="info-box__text">Description text</div> </div> </div>

Color Modifiers

  • .info-box--blue - Information
  • .info-box--sky-blue - Tips
  • .info-box--green - Success
  • .info-box--yellow - Warning
  • .info-box--orange - Alert
  • .info-box--red - Error
  • .info-box--pink - Recommendation
  • .info-box--black - Note

Size Modifiers

  • .info-box--sm - Small size
  • Default (no modifier) - Medium size
  • .info-box--lg - Large size

Style Modifiers

  • .info-box--outlined - Transparent background with border
  • .info-box--no-icon - Hide the icon

Optional Elements

  • .info-box__title - Optional title (can be omitted)
  • .info-box__icon - Can be hidden with --no-icon modifier

Usage Guide

How to use the button component

Base Class

.btn - Required base class

Variant Modifiers

  • .btn--filled - Solid background (default)
  • .btn--stroked - Outline style

Size Modifiers

  • .btn--sm - 36px height
  • .btn--md - 44px height (default)
  • .btn--lg - 52px height
  • .btn--xl - 60px height

Color Modifiers

  • .btn--blue
  • .btn--sky-blue
  • .btn--green
  • .btn--yellow
  • .btn--orange
  • .btn--red
  • .btn--pink
  • .btn--black

Special Modifiers

  • .btn--icon - Icon-only square button
  • .btn--disabled or disabled - Disabled state
